运算符

算数运算符

运算符 描述
+ 相加
- 相减
* 相乘
/ 相除
% 求余

注意： ++ 和 --是单独的语句，不是运算符

关系运算符

运算符 描述
== 是否相等
!= 是否不等
> 是否大于
< 是否小于
>= 是否大于等于
<= 是否小于等于

逻辑运算符

运算符 描述
&& 与逻辑运算
|| 或逻辑运算
! 非逻辑运算

位运算符

运算符 描述
& 按位与
| 按位或
^ 按位异或
<< 左移n位
>> 右移n位

赋值运算符

运算符 描述
= 赋值运算符
+= 相加后赋值
-= 相减后赋值
*= 相乘后赋值
/= 相除后赋值
%= 取模后赋值
<<= 左移后赋值
>>= 右移后赋值
&= 按位与后赋值
|= 按位或后赋值
^= 按位异或后赋值
