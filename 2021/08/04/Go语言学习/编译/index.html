编译

go build编译go文件

  1. 在项目目录下执行go build
  2. 在其他地方执行go build src后的路径
  3. go build -o xxx.exe取名

go run直接运行go文件

go install编译go文件并将编译结果移动到bin

交叉编译

Go支持跨平台编译

比如：在windows平台编译一个能在linux平台执行的可执行文件
