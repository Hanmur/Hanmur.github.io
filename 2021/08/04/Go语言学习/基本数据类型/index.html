整型

整型可以分为两个大类

有符号整型 无符号整型
int8 uint8
int16 uint16
int32 uint32
int64 uint64

其中uint8对应byte型，int16对应short型，int64对应long型

特殊整型

类型 描述
uint 随操作系统对应32或64
int 随操作系统对应32或64
uintptr 无符号整型，用于存放指针

八进制&十六进制

二进制 八进制 十进制 十六进制
%b %o %d %x

浮点数

浮点数分为float32和float64

遵循IEEE 754标准

对应符号%f

复数

复数分为complex64和complex128

布尔值

布尔值bool 只有两个值：true和false

默认值为false

go语言中不允许将整数型强制转换成布尔型

go语言中也不允许参与数值运算

类型转换

Go语言类型转换格式：

type_name为类型，expression为表达式

举例：将整型转换为浮点型

Go不支持隐式类型转化
